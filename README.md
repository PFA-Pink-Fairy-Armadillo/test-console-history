# Test MVP For Console History

Using this library: https://github.com/lesander/console.history

Instructions

- Install npm and boot up the server
```
npm install
nodemon Server/server.js
```
- Open `index.html`.
- Open `index2.html`
- Now you should see logs from the Client and server
